## 1. 产品概述

### 1.1 产品定位

DictBox 是一款基于 Chrome Omnibox API 的轻量级字典翻译插件。用户在浏览器地址栏输入 `db` + 空格 + 单词/短语，即可获取翻译结果，无需离开当前页面。

### 1.2 目标用户

- 日常需要中英互译的开发者、学生、翻译工作者
- 希望快速查词但不想切换标签页的 Chrome 用户
- 需要词性标注（adj./n./v.）的语言学习者

### 1.3 核心价值

| 维度 | 价值 |
|------|------|
| **效率** | 地址栏即查即用，零切换成本 |
| **准确** | 词性标注 + 多义词展示 + 匹配度排序 |
| **灵活** | 3 大翻译引擎可选，13 种目标语言 |
| **免费** | 默认 MyMemory 引擎无需注册，开箱即用 |

## 2. 功能需求

### 2.1 Omnibox 翻译（P0 核心）

| 编号 | 需求 | 状态 |
|------|------|------|
| F-001 | 输入 `db` + 空格激活翻译模式 | ✅ |
| F-002 | 输入单词后在下拉列表显示翻译结果 | ✅ |
| F-003 | 结果格式：`词性 + 含义`，如 `adj. 美丽的` | ✅ |
| F-004 | 多含义时每条单独一行显示 | ✅ |
| F-005 | 结果按匹配质量排序 | ✅ |
| F-006 | 默认将其他语言翻译为中文 | ✅ |
| F-007 | 输入中文时自动翻译为英文（智能反转） | ✅ |

### 2.2 翻译引擎（P0 核心）

| 编号 | 需求 | 状态 |
|------|------|------|
| F-010 | 默认使用 MyMemory 免费 API | ✅ |
| F-011 | 支持 Google Cloud Translation API v2 | ✅ |
| F-012 | 支持 Microsoft Dictionary API（词典优先） | ✅ |
| F-013 | Microsoft 自动语言检测（/detect 端点） | ✅ |
| F-014 | Microsoft 字典无结果时降级到翻译 API | ✅ |
| F-015 | 主引擎失败自动 fallback 到 MyMemory | ✅ |

## 4. 非功能需求

| 类别 | 要求 |
|------|------|
| **性能** | Omnibox 响应时间 < 1s（缓存命中 < 50ms） |
| **可靠性** | 引擎失败自动降级，永不崩溃 |
| **安全** | API Key 仅存储在本地 chrome.storage，不上传 |
| **隐私** | 不收集任何用户数据，不追踪使用行为 |
| **兼容性** | Chrome 88+（Manifest V3 最低版本） |
| **包大小** | < 100KB（无外部依赖） |
